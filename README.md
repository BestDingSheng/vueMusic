## 项目运行

``` bash
# install dependencies  安装依赖
npm install

# serve with hot reload at localhost:8080  运行
npm run dev

# build for production with minification 打包
npm run build

```

### 技术栈

- vue2.0 + vue-router + vuex：vue全家桶
- jsonp：用于ajax请求
- vue-siwper: banner轮播
- webpack： 构建工具
- es6: 使用es6语法
- scss: css预处理

### 总结
- 在手机上运行不太流畅
- 手机上点击播放的时候需要暂停在点击播放才有声音
- 样式排版还不是很美观
